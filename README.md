# rightFOUNDATION™ — Website Prototype

A complete, responsive, static website prototype. Built as plain HTML/CSS/JS so it can be hosted directly on GitHub Pages, Dynadot hosting, or any static host — no build step required.

Sources used: rightFOUNDATION logo, "Right Foundation Base Thought" presentation, and "THE THINK RIGHT.ORG" concept document. Commercial/agency/VC content from the ThinkRight document was intentionally excluded per the content brief.

## 2. Design system

**Colours (from official logo — one strong colour per section only):**
- Blue #2E86C9 — trust, knowledge, education
- Red #E4574E — compassion, human connection
- Green #4BA455 — sustainability, growth
- Yellow #F2AF1D — hope, energy, opportunity
- Ink #1C2733 (text/dark sections), Slate #55636F (secondary text)
- Backgrounds: White #FFFFFF, Paper #FAFAF7, Mist #EAF3FA + soft tints of each logo colour

**Typography (Google Fonts):**
- Display: Bricolage Grotesque (600–700) — headings
- Body: Figtree (400–800) — text, buttons, UI

**Rules applied:** light visual system, generous white space, strong colours limited to icons/buttons/section identifiers/card accents, no dark backgrounds except the manifesto and CTA bands, no fabricated data anywhere.

## 4. Information required from Just Right before launch

**Legal / governance (blocking):**
1. Legal entity name, registration number, and jurisdiction of rightFOUNDATION
2. Trustees / board / leadership names and approval to publish
3. Approved Privacy Policy and Terms of Use text
4. Donation / tax-exemption status (do NOT enable any donation feature before this is confirmed)

**Contact & operations (blocking):**
5. Official email address and phone number
6. Office address(es), if any are to be published
7. Social media handles

**Content (before removing placeholders):**
8. Verified impact figures (communities reached, people trained, initiatives, partners)
9. Confirmed active initiatives with locations, partners, and approved results
10. Confirmed partner names/logos with written permission to display
11. Approved photography (dignity-first; no stock clichés)
12. Final approval of all wellness language by legal/regulatory review (UAE: DHA/MOHAP/DoH awareness where relevant)

**Technical (before go-live):**
13. Final domain (e.g., rightfoundation.ae) and hosting decision
14. Form backend (email service / CRM) for the 3 prototype forms
15. Analytics preference (if any) — must match the published privacy policy

## 5. How to preview / deploy
- Preview: open `index.html` in any browser.
- GitHub Pages: push the folder to a repo, enable Pages on the main branch root.
- All paths are relative — the site works from any folder or subdomain.

## 6. Content compliance status
- No invented statistics, projects, partners, offices, registrations, or testimonials.
- All unconfirmed items are visibly marked "Content pending verification" or "Verified figure required".
- Goshala/kitchen and Collective SEWA content is presented as "models we believe in", not completed Foundation projects.
- Right Wellness carries an explicit non-medical-claims note.
